# Cowork Workspace Setup

A baseline configuration for spinning up a Claude Cowork workspace for a delivery team:
which repositories to connect, sensible default roles, and a curated starting set of
shared skills.

## What it does

- Encodes the team's standard workspace layout so each new engagement starts the same.
- Pre-selects shared skills (review, summarization, scaffolding).
- Documents the role model for members and guests.

## When to use

At the start of a new engagement or when onboarding a team to Cowork. Treat it as a
starting point and tailor roles to the client's access policy.
