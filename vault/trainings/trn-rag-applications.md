# Building RAG Applications End to End

Walks the full retrieval-augmented-generation pipeline: document chunking strategies, choosing an
embedding model, indexing into a vector store, retrieval and re-ranking, and grounding the answer
with citations. Covers the failure modes — stale chunks, lost context, ungrounded answers — that
only show up once real users arrive.
