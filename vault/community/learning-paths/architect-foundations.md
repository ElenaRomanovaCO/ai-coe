# AI Solution Architecture Foundations

A four-week path for solution architects new to applied generative AI. Builds the
vocabulary and judgment to scope a first production-grade use case: how retrieval works,
when to fine-tune versus prompt, how to pick a model tier, and how to reason about cost,
latency, and evaluation from day one. Pairs with the Reference Architecture library.
