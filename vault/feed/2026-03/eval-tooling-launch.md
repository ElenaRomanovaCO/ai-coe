# Open-source eval toolkit for RAG quality

A new open-source toolkit packages common RAG evaluation metrics (faithfulness,
context precision/recall, answer relevance) behind a simple interface.

## What this means

The biggest gap in most CoE pilots is the absence of a repeatable quality measure.
A standard eval toolkit lowers the barrier to building a regression set so model and
prompt changes can be judged objectively. Radar: **trial** — wire it into one active
RAG project and make "passes the eval set" a release gate.
