# Sample Engagement Kit — Fintech Fraud-Scoring

A demo engagement kit assembled for a stage-3 financial-services client operationalizing
fraud scoring. It bundles the assets a consultant would carry into the engagement.

## Included assets (linked from the vault)

1. **Discovery** — `assets/cross-industry/1/kickoff-template-discovery-workshop.md`
2. **Solution pattern** — `assets/financial-services/3/solution-pattern-fraud-scoring-agent.md`
3. **Risk checklist** — `assets/financial-services/2/risk-checklist-model-risk.md`
4. **ROI model** — `assets/cross-industry/2/roi-template-ai-business-case.md`
5. **Regulatory context** — `regs/us-federal/ffiec/ffiec-ai-financial.md`

## How to use

Run the discovery workshop, frame the fraud-scoring agent against the solution pattern,
size value with the ROI model, and clear governance with the risk checklist and FFIEC
context before any pilot. Prompts: see `prompts/ai-risk-assessment.md` and
`prompts/executive-briefing.md`.

This kit is illustrative demo content — no real client data.
