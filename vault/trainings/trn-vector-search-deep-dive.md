# Vector Search Deep Dive

A focused course on the retrieval half of RAG: how approximate nearest-neighbor search works,
cosine vs dot-product similarity, hybrid keyword + vector search, metadata filtering, and the
levers that move retrieval quality. Useful when a RAG pilot's answers are "almost right" and you
need to debug the retrieval, not the model.
