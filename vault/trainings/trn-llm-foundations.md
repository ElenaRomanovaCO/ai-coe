# Large Language Models: From Zero to Working Intuition

A gentle, example-driven introduction for anyone new to generative AI. Builds the mental model
you need before picking up tools: what a token is, how context windows constrain a prompt, why
models hallucinate, and how embeddings turn text into something searchable. No math-heavy theory —
just the working intuition a consultant needs to scope a first use case credibly.
